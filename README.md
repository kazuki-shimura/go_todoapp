# go_todoapp

    iniファイルを読み込むための設定
    go get "gopkg.in/go-ini/ini.v1"
    参考：https://qiita.com/geniusmaaakun/items/d2692d3eb5dd286a06a9

    SQLiteを使用するためのコマンド
    go get github.com/mattn/go-sqlite3

    sqliteのファイルに入る
    sqlite3 webapp.sql
